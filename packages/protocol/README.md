# Orchestra Protocol

Shared contracts for backend and desktop.

Goals:
- Versioned payload schemas.
- Stable API event envelopes.
- Compatibility fixtures for transition-safe deployments.

Current schemas:
- `schemas/v1/state.response.schema.json`
- `schemas/v1/refresh.response.schema.json`
- `schemas/v1/issue.response.schema.json`
- `schemas/v1/workspace.migrate.response.schema.json`
- `schemas/v1/workspace.migration.plan.response.schema.json`

Current type stubs:
- `types/v1.ts` (TypeScript)
- `types/v1.go` (Go)
